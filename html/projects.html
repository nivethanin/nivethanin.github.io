PROJECTS

This personal website was made by me from scratch! It was made using HTML, CSS, Javascript (with some JQUERY), and some PHP for email handling. You can see the source code for this on my GitHub. This was made as a final project for my web development course at Algoma University.

The Salt Tacker is a project I actually completed as I was learning javascript for said web development course. It parses transcripts made by speech language pathologists during patient examinations and tracks several attributes of text and processes the numbers into scores that are used for patient diagnosis. It is fully functional and is being used by speech language pathology graduate students at the University of Toronto currently. I have it linked in the header bar here, so please check it out! It runs on javascript. Source code available on GitHub.

After sending in the Salt Tracker, I am currently developing a quiz for a lab in the University of Toronto' speech language pathology department. The website is not up yet and source code will not be made available until the tests are complete. Currently we are building it on the react framework using and agile methodology.
